# Fast Port Scanner

## Description

A multithreaded port scanner developed using Python. The tool performs fast network scans by using threads to check multiple ports simultaneously.

## Features

* Multithreaded scanning
* Faster execution
* Open port detection
* Efficient scanning
* Command-line interface

## Technologies Used

* Python 3
* Threading Module
* Socket Library
* Kali Linux

## How to Run

```bash
python3 fast_scanner.py
```

## Learning Outcomes

* Multithreading
* Concurrent programming
* Network scanning
* Performance optimization
